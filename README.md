## What this project does

This project helps you:

- Add and manage **Spare Parts**
- Add and manage **Suppliers**
- Link parts with suppliers
- View lists of parts and suppliers through a clean UI

---

##  Tech Stack

- Java (Spring Boot)
- Spring MVC
- Spring Data JPA
- MySQL (or H2 if not configured)
- Thymeleaf (for frontend)
- Maven
